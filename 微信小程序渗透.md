# 微信小程序渗透



## Fiddler转burp

说明：由于burp抓不到小程序，fiddler没办法操作流量，所以选择转发流量，也可以在转发到其他工具，比如yakit

#### 配置转发

1、tools-options-geteway，选择手动代理，添加burp的代理端口（127.0.0.1:8080）

#### 注意事项

1、需要先打开fiddler，再打开burp

2、如果配置没问题，但是burp没有流量，可以尝试管理员权限打开fiddler





## Proxifier转burp

说明：proxifier更好用，但是有的环境不让用，只能用fiddler转

#### 添加代理服务器

1、在profile中选择proxy servers，添加burp地址（127.0.0.1），端口（8080）

#### 设置规则

1、设置application，在任务管理器中寻找微信小程序（WeChatAppEx），打开文件所在位置，复制地址，添加到application中，或者	在Browse中选择wechatappex.exe 。

2、设置action，选择添加的代理服务器（burp），点击ok后，勾选

3、设置location和default为direct（直连）模式，点击ok后保存

#### 确认burp接受到微信小程序的流量包

## E0e1-wx为微信小程序添加hook

#### 注意事项

1、使用e0e1-wx需要使用指定版本的微信（有封号风险）

2、共三个大功能，1是启动时添加 - hook，可以给微信小程序添加hook

​				 2是可以直接扫面打开的微信小程序，将源码反编译处理并保存

​				 3是可以扫描小程序，直接将泄露的后台地址，key，链接整理成excel

#### 安装E0e1

1、选择微信版本；微信旧版本下载地址&版本推荐

```
下载地址：	https://github.com/tom-snow/wechat-windows-versions/releases?page=2
微信推荐版本：WeChatSetup-3.9.9.43
```

2、安装步骤

```
教程地址：https://github.com/eeeeeeeeee-code/e0e1-wx
```

3、注意事项

```
1、下载环境时，requirements.txt中有的下载不下来，单独pip即可
2、最好直接安装对应的微信版本，上面推荐的版本内存码是8555，如果出现更换微信内存码不变（还是14xxx），可以卸载微信后，
	删除 C:\Users\Administrator\AppData\Roaming\Tencent\xwechat & WeChat。再重新安装3.9.9.43的微信
```

4、使用

~~~
1、python e0e1-wx.py	#启动，可以添加 -hook参数，实现对微信小程序添加f12（点击三个点后可以看到）。
2、启动后自动解析小程序，生成result，其中的proess.xlsx 整理出了接口，后台地址，硬编码的key
~~~
